Grundlæggende Web

I tema 2: grundlæggende web fik vi et mere grundlæggende funtament for uddannelsen og stiftede første bekendskab med kodning. Her lærte vi at bruge HTML og CSS, og efterfølgende at ligge vores produkt op på vores domæne med Filezilla. Vi lærte om udtrykket: “mobile-first”, grid, flex og media-quariers. Vi fik mere viden om de grundlæggende design principper, lærte om getaltlovene og prøvede kræfter af med styletile, moodboard, wireframes og generelle redskaber til ideudvikling. I dette team udviklede vi også vores studiestartsprøve.
